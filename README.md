# Shopper Spectrum - Customer Segmentation

## 📌 Overview
This project focuses on customer segmentation using machine learning techniques to analyze shopper behavior and purchasing patterns.

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Clustering for Customer Segmentation
- Data Visualization

## 🛠 Technologies Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## 📊 Outcome
Identified different customer segments based on spending behavior to help improve targeted marketing strategies.
